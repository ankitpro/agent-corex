📖 Quick Setup Guide

Follow these steps to get Agent-Corex running locally and test it with this dashboard. All commands are ready to copy - click the copy button next to each command!

1 Clone the Repository

First, clone the Agent-Corex repository from GitHub to your local machine.

git clone https://github.com/ankitpro/agent-corex.git cd agent-corex
2 Create Virtual Environment

Create a Python virtual environment to keep dependencies isolated.

python -m venv venv source venv/bin/activate
python -m venv venv venv\Scripts\activate
3 Install Dependencies

Install Agent-Corex and all required dependencies using pip.

pip install -e .

ℹ️ This installs Agent-Corex in development mode, so you can test it locally.

4 Start the Backend

Launch the FastAPI server that runs on your local machine.

uvicorn agent_core.api.main:app --reload

✅ Server should start on http://localhost:8000
📖 API docs available at http://localhost:8000/docs

5 Connect Dashboard

Go to the "Connection" tab below and click "Test Connection" to verify your backend is running.

✅ All set! Once your backend is running, you'll be able to test tool retrieval in the Connection tab.
